# Taxi Demand Forecasting

Forecasting hourly NYC taxi demand using machine learning and MLOps practices.

## Overview

This project predicts hourly taxi demand in New York City using historical trip data. The goal is to build an end-to-end machine learning system that covers data validation, feature engineering, model training, experiment tracking, API development, containerization, CI/CD, and cloud deployment.

## Tech Stack

* Python
* Pandas
* Scikit-learn
* Great Expectations
* MLflow
* FastAPI
* Docker
* GitHub Actions
* AWS ECS (Fargate)
